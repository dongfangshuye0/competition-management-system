# README 项目说明完善

## 目标

- 完善根目录 README，让项目可以被快速理解、安装、启动和维护。

## 修改范围

- `README.md`

## 核心实现

- 补充项目简介、功能概览、技术栈和目录结构。
- 补充数据库初始化、前后端启动、默认账号和常用命令。
- 补充关键环境变量、主要页面、项目文档索引和常见问题。

## 影响范围

- 仅影响项目文档。
- 不修改前端、后端、数据库脚本或运行逻辑。

## 验证方式

- 读取 README 内容，检查 Markdown 结构、中文编码和命令示例。

## 已知限制

- README 中的运行配置基于当前项目默认配置，部署到生产环境时仍需按实际服务器、数据库和域名调整。
