# 首页底部信息栏

## 目标

- 在首页底部增加类似参考图的深色站点信息栏。
- 展示平台标识、底部导航、版权备案信息和官方二维码区域。

## 修改范围

- `frontend/src/views/HomeView.vue`
- `frontend/src/styles.css`

## 核心实现

- 在首页内容末尾新增 `portal-footer` 页脚区域。
- 页脚采用左侧标识、中间站点信息、右侧二维码区域的三列布局。
- 在窄屏下改为纵向居中布局，避免文字和二维码溢出。
- 二维码区域先使用 CSS 视觉占位，便于后续替换为真实公众号二维码图片。

## 影响范围

- 仅影响首页底部展示。
- 不修改竞赛列表加载、登录入口、报名入口和后端接口。

## 验证方式

- 运行前端构建命令，检查 Vue 模板和 TypeScript 类型。
- 访问首页并滚动到底部，检查页脚是否正常显示。
- 检查窄屏下页脚是否自动换行且无明显溢出。

## 已知限制

- 备案号和二维码素材当前没有真实数据，页面使用“待配置”和二维码占位视觉。
