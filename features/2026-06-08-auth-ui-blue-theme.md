# 登录注册页与蓝色主题优化

## 目标

优化 Vue3 前端登录和注册入口的视觉体验，并将系统主导航调整为蓝色主题。

## 修改范围

- `frontend/src/App.vue`
- `frontend/src/views/LoginView.vue`
- `frontend/src/styles.css`

## 核心实现

- 将登录页改为左右分栏结构，左侧展示蓝色品牌区和业务入口标签，右侧提供登录/学生注册切换表单。
- 增加学生注册表单，调用现有 `/api/auth/register` 接口，注册成功后自动登录进入工作台。
- 将系统侧边导航调整为蓝色渐变，并补充品牌标识和角色标签。
- 增加桌面、平板、手机断点，注册表单在窄屏下自动单列，避免横向溢出。

## 影响范围

- 前端登录、注册和登录后的整体导航视觉。
- 不改变后端认证接口和数据库结构。

## 验证方式

- 在 `frontend` 执行 `npm run build`，类型检查和 Vite 构建通过。
- 在浏览器打开 `http://localhost:5173/login`，验证登录页蓝色分栏正常显示。
- 切换到学生注册表单，验证字段展示完整且无横向溢出。
- 使用管理员演示账号登录，验证进入工作台后侧边导航为蓝色。

## 已知限制

- 当前前端入口为 Vite 服务 `http://localhost:5173`；`http://localhost:8080` 为后端 API 服务，不再作为前端首页入口。
- Vite 构建仍提示主包体积较大，后续可按路由拆包优化。
