# 高校竞赛管理系统需求文档
## 1. 项目背景
为了解决高校竞赛管理中存在的报名流程繁琐、信息通知不及时、作品管理混乱等问题，开发一个基于Spring Boot+Vue的高校竞赛管理系统，并集成DeepSeek大模型提供智能化服务。

## 2. 用户角色
- 学生：普通用户，可以浏览竞赛、报名参赛、提交作品
- 指导教师：可以审核学生报名、查看所带队伍信息
- 系统管理员：拥有系统所有权限，可以管理用户、竞赛、作品等

## 3. 核心功能清单（✅必做，❌不做，🔆亮点）
### 3.1 学生端功能
✅ 注册、登录、修改个人信息（姓名、学号、专业、班级、手机号）
✅ 浏览竞赛列表、查看竞赛详情
✅ 在线报名竞赛（上传报名表）
✅ 提交竞赛作品（上传文件）
✅ 查看报名状态、作品评审结果
✅ 接收系统通知
🔆 智能答疑助手（DeepSeek）
🔆 竞赛智能推荐（DeepSeek）

### 3.2 教师端功能
✅ 查看所指导的学生报名信息
✅ 审核学生的竞赛报名申请
✅ 查看学生提交的作品
✅ 给作品打分和写评语
✅ 接收系统通知

### 3.3 管理员端功能
✅ 用户管理（增删改查学生、教师账号）
✅ 竞赛管理（发布、修改、删除竞赛信息）
✅ 报名管理（查看所有报名信息）
✅ 作品管理（查看所有作品、分配评审教师）
✅ 成绩管理（公布成绩、生成排名）
✅ 通知管理（发布系统通知）

## 4. DeepSeek亮点功能（只选2个！）
1. 智能答疑助手：解答学生关于竞赛报名、规则、提交等常见问题
2. 竞赛作品智能初评：对学生提交的论文/报告类作品进行自动评分和评语生成

## 5. 非功能需求
- 系统响应时间：页面加载时间不超过3秒
- 兼容性：支持Chrome、Edge、Firefox浏览器
- 安全性：用户密码加密存储，接口需要身份认证
